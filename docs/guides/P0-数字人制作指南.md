# P0 — 数字人制作指南（团队成员真人克隆）

> 预计耗时: 录制30分钟 + 等待训练24-48小时

## 推荐平台: 硅基智能 DUIX

- 官网: https://duix.com 或 https://guiji.ai
- 手机号注册即可使用
- 定制克隆: 约 9800元/套（形象+声音）
- 快速克隆: 最快1秒视频即可生成（效果相对简单）

## Step 1: 录制真人素材

### 录制要求清单（请团队成员严格执行）

| 项目 | 要求 |
|------|------|
| **时长** | 3-5分钟连续说话视频 |
| **方向** | 竖屏拍摄（手机竖着拿） |
| **分辨率** | 1080P 以上 |
| **背景** | 纯色墙壁或绿幕，无杂物 |
| **灯光** | 正面打光，均匀无阴影，避免逆光 |
| **着装** | 穿直播时的衣服，上半身整洁 |
| **妆容** | 淡妆即可，与直播时一致 |
| **表情** | 自然说话，偶尔微笑，适当手势 |
| **语速** | 正常偏慢，吐字清晰 |
| **视线** | 看镜头，模拟面对观众 |
| **环境** | 安静，无回声，无背景噪音 |

### 说话内容参考（随便讲，以下仅供参考）

```
大家好，欢迎来到我们的直播间！
我是你们的主播XXX，今天给大家带来一款特别棒的产品。
这个产品是来自浙江奉化的桑葚汁，口感非常好。
我平时特别喜欢喝果汁，尤其是这种天然的。
大家如果有什么问题可以在弹幕里问我哦。
觉得好的点点关注，不迷路！
好的，感谢大家的支持，我们下次再见！
（以上内容反复说即可凑够3-5分钟）
```

### 录制技巧

1. 手机固定在三脚架上，不要手持
2. 镜头高度与眼睛平齐
3. 人物在画面中占比约 60-70%（头顶留一点空间）
4. 录之前先试录10秒看效果
5. 多录几段取最好的一段

## Step 2: 上传硅基智能训练

1. 访问 https://duix.com 或 https://guiji.ai
2. 手机号注册登录
3. 选择「数字人克隆」或「形象定制」
4. 上传录制好的视频
5. 按提示完成声纹采集（可能需要朗读指定文字）
6. 提交训练，等待 24-48 小时

## Step 3: 测试数字人

训练完成后：
1. 在平台后台输入一段测试文字
2. 检查口型是否同步
3. 检查声音是否自然
4. 检查表情是否生动
5. 如不满意，可调整参数或重新录制

## Step 4: 对接 OBS 推流

1. **安装 OBS Studio**: https://obsproject.com/download
2. **获取数字人画面**: 硅基智能提供的推流地址或浏览器源
3. **OBS 配置**:
   - 来源 → 添加「浏览器」或「媒体源」→ 填入数字人画面地址
   - 添加产品展示图/视频作为背景
   - 设置合适的画面布局（数字人在左/右，产品在另一侧）
4. **推流到抖音**:
   - 抖音APP → 开直播 → 选择「电脑直播」→ 获取推流地址
   - OBS → 设置 → 推流 → 填入抖音的推流地址和密钥
   - 点击「开始推流」
