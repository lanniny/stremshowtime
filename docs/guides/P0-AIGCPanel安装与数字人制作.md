# AIGCPanel 安装与数字人制作 — 保姆级教程

> 工具: AIGCPanel v1.4.0（开源免费）
> 系统: Windows 10+ / Mac / Linux
> 预计耗时: 安装30分钟 + 克隆制作1-2小时

---

## 第一步: 下载安装 AIGCPanel

### 1.1 下载

打开浏览器访问: **https://aigcpanel.com/zh/download**

选择对应版本:
- Windows 电脑 → 点击 **「64位芯片」**（绝大多数电脑选这个）
- Mac 电脑 → 根据芯片选 Intel 或 Apple

### 1.2 安装

1. 双击下载的安装包
2. 一直点**「下一步」**直到安装完成
3. 桌面出现 AIGCPanel 图标

### 1.3 首次启动

1. 双击打开 AIGCPanel
2. 软件会提示**下载模型**
3. 点击下载，等待模型下载完成（可能需要10-30分钟，取决于网速）
4. 下载完成后，模型状态显示为**绿色**即可使用

---

## 第二步: 声音克隆（让数字人说你的声音）

### 2.1 录制声音素材

用手机录一段自己的声音:
- **时长**: 5-10秒即可（说一段完整的话）
- **环境**: 安静，无回声，无背景噪音
- **音量**: 适中，不要太大导致失真
- **语速**: 正常稳定，不要有明显停顿
- **语言**: 说普通话（或目标语言）

**录制内容参考**:
```
大家好，欢迎来到我们的直播间，我是你们的主播小桑，
今天给大家带来一款来自浙江奉化的桑葚汁，特别好喝！
```

### 2.2 在 AIGCPanel 中克隆

1. 打开 AIGCPanel → 进入**「声音克隆」**模块
2. 点击**「添加音色」** → 上传刚录制的声音文件
3. 选择一个**运行中的模型**（绿色状态的）
   - 推荐模型: GPT-SoVITS 或 IndexTTS
4. 选择刚上传的音色
5. 输入一段测试文字（如"你好，测试一下声音效果"）
6. 点击**「开始克隆」**
7. 等待完成 → 播放试听效果
8. 效果满意就可以用了！

## 第三步: 数字人形象制作

### 3.1 录制形象视频

这是关键步骤！视频质量直接决定数字人效果。

**录制要求清单**:

| 项目 | 要求 |
|------|------|
| 方向 | **竖屏**（手机竖着拍） |
| 分辨率 | 1080P 以上 |
| 时长 | 1-3分钟连续说话 |
| 背景 | **纯色墙壁**（白/灰/绿，无杂物） |
| 灯光 | 正面打光，均匀无阴影 |
| 着装 | 穿直播时的衣服 |
| 表情 | 自然说话，偶尔微笑 |
| 视线 | **看镜头**（模拟面对观众） |
| 头部 | 保持基本不动，嘴部动作清晰 |
| 环境 | 安静，无背景噪音 |

**重要提示**:
- 手机固定在三脚架上，不要手持
- 人物在画面中占比约 60-70%
- 头顶留一些空间
- 录之前先试录10秒看效果

### 3.2 在 AIGCPanel 中制作数字人

1. 打开 AIGCPanel → 进入**「视频合成」**模块
2. 上传录制好的形象视频
3. 选择之前克隆好的声音
4. 输入要说的文字（可以粘贴直播脚本）
5. 点击**「合成」**
6. 等待处理完成
7. 预览效果 — 检查口型是否同步、声音是否自然

## 第四步: 数字人直播（推流到抖音）

### 4.1 准备工作

1. **下载抖音直播伴侣**: https://streamingtool.douyin.com/
2. 安装并登录抖音账号
3. 确保账号有直播权限

**注意**: 
- 不要在多个设备同时登录同一账号
- 手机端不要频繁切换账号（会触发平台保护机制导致断连）

### 4.2 开播配置

**方式一: 通过抖音直播伴侣（推荐）**

1. 打开 AIGCPanel → 进入**「数字人直播」**模块
2. 选择制作好的数字人形象
3. 选择克隆好的声音
4. AIGCPanel 会创建一个虚拟摄像头
5. 打开**抖音直播伴侣** → 添加来源 → 选择摄像头 → 选**「HD Camera」**
6. 直播伴侣会自动获取 AIGCPanel 的数字人画面和声音
7. 搭建直播间（添加产品背景、小黄车等）
8. 点击**「开始直播」**

**方式二: 通过 OBS Studio**

1. AIGCPanel 开启数字人 → 选择**窗口模式**
2. 打开 OBS → 来源 → 添加**「窗口捕获」** → 选择 AIGCPanel 窗口
3. OBS → 设置 → 推流 → 填入抖音推流地址
4. 点击**「开始推流」**

### 4.3 直播互动配置

AIGCPanel 支持直播互动功能:
1. 在直播模块中开启**「直播互动」**
2. 连接到抖音直播间（粘贴直播间链接）
3. 系统会自动读取弹幕并生成回复
4. 数字人会自动播报回复内容

**与 OpenClaw 配合**:
- AIGCPanel 负责: 数字人形象驱动 + TTS语音播报
- OpenClaw 负责: 弹幕智能分类 + 回复内容生成 + 投诉通知

## 第五步: 商品话术自动生成（AIGCPanel内置功能）

如果通过抖音直播伴侣开播:
1. 登录抖音账号
2. 选择**「导入抖音商品」**
3. 点击确认商品
4. 系统会**自动生成商品直播话术**
5. 可手动修改或使用 OpenClaw Skill 1 生成更专业的脚本

## 硬件要求

| 配置项 | 最低要求 | 推荐配置 |
|--------|---------|---------|
| CPU | 4核以上 | 8核以上 |
| 内存 | 8GB | 16GB |
| 显卡 | 无GPU也可跑（CPU模式） | NVIDIA GTX 1060+ |
| 硬盘 | 20GB 空闲 | 50GB+ SSD |
| 网络 | 上行 5Mbps | 上行 10Mbps+ |

**说明**: 
- 有 NVIDIA 显卡效果更好、速度更快
- 没有显卡也能用 CPU 模式，只是合成速度慢一些
- 直播时建议关闭其他占资源的软件

## 常见问题

**Q: 安装后打不开？**
A: 右键以管理员身份运行试试

**Q: 模型下载很慢？**
A: 可能需要科学上网，或等低峰期下载

**Q: 数字人口型不同步？**
A: 检查录制视频是否稳定，嘴部动作是否清晰。重新录一段试试

**Q: 直播伴侣找不到摄像头？**
A: 确保 AIGCPanel 的虚拟摄像头已开启，重启直播伴侣

**Q: 需要付费吗？**
A: AIGCPanel 基于 Apache-2.0 开源，完全免费！
