# P0 — 抖音弹幕抓取工具选型与配置

> 预计耗时: 1-2小时配置

## 推荐方案对比

| 方案 | 难度 | 费用 | 适合 |
|------|------|------|------|
| **BarrageGrab** | ⭐⭐ | 免费开源 | 首选推荐 |
| 抖音弹幕姬 dycast | ⭐⭐ | 免费开源 | 备选 |
| TikHub API | ⭐ | 付费 | 最简单但要花钱 |
| 来播/播塔 | ⭐ | 200-800元/月 | 商业工具，最省心 |

## 方案A（首选）: BarrageGrab

开源全平台弹幕抓取工具，支持抖音、快手、B站等。

**GitHub**: https://github.com/ape-byte/DouyinBarrageGrab

### 安装步骤（在服务器上）

```bash
# 1. 下载 BarrageGrab
git clone https://github.com/ape-byte/DouyinBarrageGrab.git
cd DouyinBarrageGrab

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置直播间
# 编辑配置文件，填入抖音直播间 URL 或房间号

# 4. 启动抓取
python main.py

# 5. 弹幕数据通过 WebSocket 输出，可转发到 OpenClaw
```

### 与 OpenClaw 对接思路

```
BarrageGrab (抓取弹幕)
    ↓ WebSocket / 文件输出
OpenClaw Skill 2 (弹幕分类+回复)
    ↓ 回复文本
数字人 TTS 播报 / 直播间公屏回复
```

## 方案B（备选）: 抖音弹幕姬 dycast

更轻量，输入房间号即可获取弹幕。

**GitHub**: https://github.com/skmcj/dycast

### 特点
- 只需输入房间号即可实时获取弹幕
- 可将弹幕转发到自己的后端服务
- 前端可视化界面

## 方案C（最简单）: 商业工具

如果不想折腾技术：

1. **来播助手** — 搜索"来播"下载，提供弹幕监控和数据导出
2. **播塔** — 搜索"播塔"，提供弹幕互动和直播辅助
3. **抖音直播伴侣** — 抖音官方工具，有基础弹幕显示

商业工具的优势是开箱即用，但 API 对接能力有限。
MVP 阶段可以先用商业工具 + 手动复制弹幕到 OpenClaw 的半自动模式。

## MVP 阶段建议

**先用半自动模式跑通流程**：

1. 打开弹幕工具（任一方案），看到弹幕流
2. 运营人员挑选需要回复的弹幕
3. 复制弹幕内容发送到 OpenClaw（通过飞书/终端）
4. OpenClaw Skill 2 自动分类并生成回复
5. 运营人员将回复粘贴到直播间 / 输入数字人后台

**验证效果后再做全自动化**。
